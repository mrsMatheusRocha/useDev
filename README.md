# useDev
> A vitrine interativa — um clone de e-commerce elegante, rápido e feito para inspirar.

Bem-vindo ao useDev — não é só um conjunto de hooks: é uma experiência visual que funciona como uma loja demo. Pense neste projeto como a vitrine perfeita para testar interfaces, UX de compras, animações e padrões modernos de front-end. Explore o site como se estivesse navegando em uma loja real: catálogo, busca, carrinho, checkout simulado e páginas de produto com comportamento real.

Por que vale a pena olhar
- Experiência pronta: navegue como cliente — catálogo, filtros, produto, carrinho e checkout.
- Visual moderno: design limpo, responsivo e com microinterações.
- Ideal para demonstrações: ótimo para mostrar habilidades, protótipos ou inspirar seu próximo projeto.
- Documentação viva: o site é a melhor forma de conhecer o que o projeto entrega.

Destaques da "loja" demo
- Página inicial com destaques e produtos em destaque
- Busca e filtros básicos para encontrar produtos rapidamente
- Página de produto com imagens, descrição e variações
- Carrinho persistente (simula o comportamento real de um e‑commerce)
- Fluxo de checkout simulado — perfeito para testar telas e validações
- Layout responsivo para dispositivos móveis e desktop

git clone https://github.com/mrsMatheusRocha/useDev.git
Abra o site localmente com seus scripts de preferência (há scripts básicos para iniciar o ambiente de desenvolvimento).

Para quem é este projeto
- Desenvolvedores que querem um exemplo bonito de e‑commerce para aprender UI/UX
- Designers que desejam ver componentes em ação
- Times que precisam de uma base visual para apresentar ideias a clientes
- Criadores de conteúdo que querem gravar demos ou vídeos com um front-end real

Como usar este projeto na prática
- Inspire-se nas telas e comportamento
- Reaproveite componentes e padrões visuais
- Teste integrações (pagamento fake, analytics, A/B)
- Use como portfólio: personalize conteúdos e mostre suas alterações
